# ¿Qué buscar para el Sonido (SFX)?

Para que el audio sea "Chingón", busca sonidos con estas etiquetas en librerías como Freesound.org o Sonniss:

1. **Interfaz/UI:**
   - *Mechanical Keyboard clicks*: Busca sonidos cortos y "clackies".
   - *Digital UI beep*: Sonidos agudos muy cortos (blips).
   - *Data stream / Hissing*: Ruido blanco suave para cuando se carga texto.

2. **Ambiente:**
   - *Server room hum*: Un zumbido de baja frecuencia constante.
   - *Electronic interference*: Estática aleatoria suave.

3. **In-Game:**
   - *Clock ticking (high speed)*: Para cuando el temporizador esté en rojo.
   - *Glitch/Distortion*: Para cuando el jugador comete un error.
