# Humano - Witness.me (Atestiguame)

## Descripción del proyecto
**Witness.me (Atestiguame)** es una experiencia de terror inmersiva desarrollada en Flutter que combina narrativa oscura con mecánicas de juego. El proyecto utiliza un motor de juego (Flame) para manejar interacciones dinámicas y se integra con servicios de Firebase para la gestión de usuarios y persistencia de datos. Incluye una atmósfera tensa con soporte de video, audio espacial y sistemas de pago integrados para contenido adicional.

## Cómo ejecutarlo
1.  Asegúrate de tener instalado el SDK de Flutter (versión 3.9.2 o superior).
2.  Clona este repositorio o navega hasta la carpeta del proyecto.
3.  Ejecuta `flutter pub get` para instalar las dependencias.
4.  Conecta un emulador o dispositivo físico.
5.  Ejecuta el proyecto con:
    ```bash
    flutter run
    ```
