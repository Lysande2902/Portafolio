## ✅ VENTAJAS DE FUSIONAR

### 1. **Menos Mapas = Menos Trabajo**
- 4 mapas en vez de 7
- Menos diseño de niveles
- Menos testing de navegación

### 2. **Más Variedad por Arco**
- 2 enemigos = más dinámico
- Cambio de mecánica a mitad del arco
- Más interesante para el jugador

### 3. **Narrativa Más Rica**
- 2 víctimas por arco = más historia
- Conexiones temáticas (Gula+Avaricia = excesos)
- Más impacto emocional

### 4. **Menos Assets Faltantes**
- Solo necesitas 3 imágenes fusionadas + 1 de Ira
- En vez de 5 imágenes separadas

### 5. **Testing Más Rápido**
- 4 arcos en vez de 5
- Menos combinaciones de bugs
- Más tiempo para pulir

```dart
// En BaseArcGame
int currentPhase = 1; // 1 o 2
Enemy? currentEnemy;

void onEvidenceCollected() {
  evidenceCount++;
  
  if (evidenceCount == 5 && currentPhase == 1) {
    // Checkpoint alcanzado
    switchToPhase2();
  }
  
  if (evidenceCount == 10 && currentPhase == 2) {
    // Todas las evidencias recolectadas
    openExitDoor();
  }
}

void switchToPhase2() {
  currentPhase = 2;
  // Remover enemigo 1
  remove(enemy1);
  // Spawner enemigo 2
  add(enemy2);
  currentEnemy = enemy2;
}
```
