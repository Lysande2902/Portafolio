## 🔴 ARCO 1: GULA (The Weight of Consumption)

### 📊 Estadísticas
- **Tema**: Horror psicológico - Consumo oscuro
- **Enemigo**: Sabueso de la Gula
- **Mecánica Principal**: Weight System
- **Duración**: 5-8 minutos
- **Mapa**: 2400x1600

### 🎮 Cómo Jugar
```
1. Recoge evidencia (5 items)
2. Cada evidencia = +peso (-velocidad, +poder)
3. Con 5+ evidencia: ¡Puedes disparar comida!
4. Distrae al sabueso con comida
5. Haz tu camino al exit door

Estrategia:
- Rápido sin evidencia → Lento con todo
- Pero tienes poder de fuego
- Balance es clave
```

### 📍 Puntos Clave
- **Inicio**: (200, 800)
- **Exit**: (2200, 800)
- **Enemigo** waypoints: 8 ubicaciones
- **Evidencia**: 5 items distribuidos
- **Escondites**: 6 spots estratégicos

## 😴 ARCO 6: PEREZA (Hospital of Gray)

### 📊 Estadísticas
- **Tema**: Hospital gris - Sigilo vs Ruido
- **Enemigo**: Babosa Durmiente (Dormida pero vigilante)
- **Mecánica Principal**: Noise Detection
- **Duración**: 8-10 minutos
- **Mapa**: 2400x1600

### 🎮 Cómo Jugar
```
EL ENEMIGO ESTÁ DORMIDO PERO VIGILANTE

Acciones:
- Correr (Sprint):    Ruido 100% → Enemigo DESPIERTA
- Caminar (Walk):     Ruido 30%  → Enemigo activo
- Rastrear (Crawl):   Ruido 5%   → Muy sigiloso
- Esperar (Still):    Ruido 0%   → Enemigo se ADORMECE

Sistema de Sueño del Enemigo (0-100%):
- >80% dormido: NO persigue
- Esperar lo sube
- Tu ruido lo baja rápido
```

### 📍 Puntos Clave
- **Inicio**: (200, 800)
- **Exit**: (5600, 800)

- **Evidencia**: 5 items

- **Zonas Oscuras**: 8 áreas para sigilo
