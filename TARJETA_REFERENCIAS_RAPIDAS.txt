📱 FLUJO DE INTEGRACIÓN (7 PASOS)

Step 1: LECTURA INICIAL (30 min)
  └─ Todos leen: 01_INICIO_RAPIDO_LECTURA_INMEDIATA/RESUMEN_VISUAL_FINAL.txt

Step 2: IDENTIFICAR ROL (5 min)
  └─ Lee: INDICE_MAESTRO_SESION.md (encuentra tu rol)

Step 3: SETUP INICIAL (30 min)
  ├─ Backend: Lee QUICK_START_30_MIN.md
  ├─ Mobile: Lee QUICK_START_30_MIN.md
  └─ Todos: Install dependencies

Step 4: COPIAR CÓDIGO (15 min)
  ├─ Backend: Copia ENDPOINTS_FASE_1_2.js a tu proyecto
  └─ Mobile: Copia 3 archivos .dart a lib/screens/

Step 5: INTEGRAR (60 min)
  ├─ Backend: Integra endpoints en app.js
  ├─ Mobile: Conecta endpoints en Flutter
  └─ Todos: Sigue COPY_PASTE_CHECKLIST.md

Step 6: COMPILAR Y TESTEAR (30 min)
  ├─ Backend: npm test + test-api-completo.js
  ├─ Mobile: flutter run (device real)
  └─ QA: Testea endpoints y UI

Step 7: VALIDAR (15 min)
  ├─ Portfolio upload 100% funcional
  ├─ Ratings guardando en BD
  ├─ No crashes críticos
  └─ Listo para próxima feature
