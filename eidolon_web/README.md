# Instituto Eidolon | Innovación Neural

## Descripción del proyecto
El sitio web del **Instituto Eidolon** es una pieza narrativa interactiva que forma parte del universo de *Witness.me*. Presenta la fachada de una institución médica de vanguardia dedicada a la innovación neural y la edición del dolor. El sitio cuenta con micro-animaciones, efectos cinematográficos y una transición desde un falso error 404 hacia el sitio real, creando una atmósfera de misterio y suspenso.
